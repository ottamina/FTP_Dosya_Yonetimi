FTP Dosya Yonetim Paneli - GitHub'siz kurulum

1. Bu klasoru USB'den bilgisayarinizda kalici bir yere kopyalayin.
   Ornek: C:\FTP-Manager
2. Docker Desktop'u kurun ve acin.
3. Baslat.bat dosyasina cift tiklayin. USB paketindeki yerel imaj arsivi
   varsa internet gerekmeden yuklenir. Arsiv yoksa Docker Hub'dan indirilir.
4. Tarayicida acilan panelde ilk yonetici hesabini olusturun.
5. Eski verileri tasimak istiyorsaniz, once eski bilgisayardaki panelden
   "Yedegi indir" secenegiyle ZIP dosyasini indirin. Yeni panelde
   "Yedekten geri yukle" seceneginden bu ZIP'i yukleyin.

Bu klasor GitHub veya Git gerektirmez. Baslat.bat guncel Docker Hub
imajlarini indirir ve uygulamayi baslatir.

Notlar:
- Durdur.bat sadece uygulamayi durdurur; verileri silmez.
- Durum.bat calisan servisleri gosterir.
- Port cakismasi olursa .env dosyasindaki portlari degistirin.
- .env dosyasi yereldir; token veya parola varsa kimseyle paylasmayin.
